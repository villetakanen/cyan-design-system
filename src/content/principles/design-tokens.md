# Design Tokens

Core Design tokens can be found at `[cyan]/src/tokens` directory. 

## Units

### Baseline grid

| Token | Value | Description |
|-------|-------|-------------|
| `--cn-grid` | `0.5rem` | The baseline grid unit, 8px |
| `--cn-gap` | `--cn-grid) * 2` | The gap unit, 16px |
| `--cn-line` | `--cn-grid) * 3` | The line unit, 24px |
